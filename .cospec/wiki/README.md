### 📋 项目概述

**项目定位**: 基于微服务架构的智能文档管理系统
**技术栈**: Go、Gin、Vue.js、PostgreSQL、Docker、gRPC、Python、Nginx、K8s
**架构特点**: 微服务架构、前后端分离、向量搜索、高可用

### 🏗️ 组织结构

LAST-doc-2/
├─ cmd/                 # 主入口：后端服务启动
│  ├─ main.go           # 程序入口，初始化应用环境
├─ internal/            # 核心模块：业务逻辑
│  ├─ handler/          # 接口层：接收HTTP请求
│  ├─ service/          # 服务层：处理业务逻辑
│  ├─ repository/       # 数据层：数据库操作
│  ├─ model/            # 数据模型：实体定义
│  ├─ middleware/       # 中间件：认证、日志
│  └─ router/           # 路由配置：API路由
├─ web/                 # 前端应用
│  └─ src/              # Vue.js前端源码
├─ python-parser-service/ # 文档解析服务
├─ k8s/                 # Kubernetes部署配置
├─ scripts/             # 部署和运维脚本
├─ docker-compose.yml   # Docker编排配置
└─ Dockerfile           # Docker镜像构建
